# Emacs Configuration

This is a minimal Emacs configuration that I personally use. I use `org` mode for the configuration, so everything can be viewed in the `config.org` file.

>[!NOTE]
> This project is not updated regularly.

## Dependencies

- [Fira Code Nerd Font](https://www.nerdfonts.com/font-downloads)
- [Doom Themes](https://github.com/doomemacs/themes)
- [Dashboard](https://github.com/emacs-dashboard/emacs-dashboard)
- [Magit](https://github.com/magit/magit)
- [Company](https://github.com/company-mode/company-mode)
- [Vterm](https://github.com/akermu/emacs-libvterm)
- [Projectile](https://github.com/bbatsov/projectile)
- [Eglot](https://github.com/joaotavora/eglot)

## Setup

### Linux/MacOS
Clone this repo as `emacs` under your config directory.
`~/.config/emacs`

### Windows
Clone this repo as `.emacs.d` under your AppData Roaming directory.
`$HOME\AppData\Roaming\.emacs.d`

>[!NOTE]
> `Vterm` does not work on Windows, so I disable that package if I am using a Windows machine.
